# Crypto Price Tracker (Python Web Scraping)

This project scrapes the **Top 10 Cryptocurrency prices** from CoinMarketCap using Python and Selenium.

## Features
✔ Scrapes real-time crypto data  
✔ Extracts price, 24h change, and market cap  
✔ Saves data to CSV and Excel files  
✔ Uses Selenium for automated web scraping  

## Technologies Used
- Python
- Selenium
- Pandas
- WebDriver Manager

## Installation

Clone the repository

git clone https://github.com/yourusername/crypto-price-tracker.git

Install dependencies

pip install -r requirements.txt

Run the script

python tracker.py

## Output

The program generates:

crypto_prices.csv  
crypto_prices.xlsx

These contain the latest cryptocurrency prices.
